# 📊 SalesVision Dashboard

A Sales Analytics Dashboard built with **React.js** and **Recharts**.

## 🔗 Live Demo
(https://vidhya-102003.github.io/sales-dashboard)

## 📸 Features
- 📈 Monthly Sales Bar Chart
- 🛒 Category-wise Pie Chart
- 🔢 KPI Cards (Sales, Revenue, Orders, Avg Order Value)
- 🔍 Filter by Product Category
- 🌙 Dark themed professional UI

## 📷 Screenshot
![Dashboard Preview](screenshots/dashboard.png)

## 🛠️ Tech Stack
- React.js
- Recharts
- CSS (Inline + External)

## 🚀 Getting Started
```bash
git clone https://github.com/vidhya-102003/sales-dashboard.git
cd sales-dashboard
npm install
npm start
```
